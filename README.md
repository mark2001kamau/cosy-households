## 📖 Project Overview
**CosyHouseholds** is a sleek, static e-commerce website designed to showcase and advertise home products. This project was built to demonstrate proficiency in **Responsive Web Design (RWD)** and **DOM Manipulation** using Vanilla JavaScript.

### 🌟 Key Features
- **Responsive Layout:** Optimized for mobile, tablet, and desktop using CSS Flexbox and Grid.
- **Interactive Product Catalog:** Dynamic filtering and "Quick View" modals powered by JavaScript.
- **Product Search:** A functional search bar that filters displayed items in real-time.
- **Smooth Navigation:** Implementated "Scroll-to-top" buttons and sticky navigation for a better user experience.
- **Contact Form Validation:** Client-side validation to ensure user data is correct before submission.

---

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom animations, transitions, and media queries (No frameworks used!).
- **JavaScript (ES6+):** Handling UI interactions and dynamic content updates.
